## Priorities

- [<] scheduling
- [?] question
- [!] important (action!)
- [*] star (key information)
- ["] quote (direct quote)
- [l] location (typically for an internal link)
- [b] bookmark (typically for an external link)
- [n] note (typically for an annotation)
- [i] information
- [S] savings
- [I] idea
- [p] pros
- [c] cons
